# My Meow

## 概要
このアプリは、キャットフードや猫用品等の情報をシェアできる愛猫家のためのウェブアプリケーションです。ユーザーが愛猫の特徴（食の好みや体調の悩み等）を登録したうえで様々な情報を投稿することで、より愛猫の参考となる情報収集ができます。

## サイトテーマ
「あの子が好きならうちの子も好きかも？」

## テーマを選んだ理由
猫に限らずですが、ペットを飼っていると他の人はどうしているんだろう、何を使っているんだろう、と思うことが多々あります。このアプリでは、実際に猫を飼っている人のみが愛猫に基づく様々な情報を投稿できるようにすることで、よりリアルな役立つ情報にアクセスできる仕組みを提供します。

### 【きっかけ】
うちの猫がおなかが弱く、どんなフードなら大丈夫なのか、同じようにおなかが弱い子は何を食べているのか等が知りたくて調べても、PR情報ばかりでなかなか欲しい情報にたどり着けなかった。その経験から、猫特化のわかりやすい情報交換サイトがあればと思った。

### 【問題点】
+ 同じ悩みを持つ人がなかなか見つけられない
+ リアルな感想が見つけづらい

### 【解決策】
+ ユーザー登録時に愛猫の特徴を登録できるようにする
+ リアルな感想のため、写真を必須にする
+ 商品名やカテゴリのほか、特徴でも検索できるようにする

## ターゲットユーザー
+ 猫を飼っている人
+ 猫に関する情報交換がしたい人

## 主な利用シーン
+ 気になっている商品のリアルな口コミが知りたい時
+ 愛猫の特徴に合った商品を探したいとき
+ 良かったものを共有したいとき
